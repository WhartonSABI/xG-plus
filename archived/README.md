# Archived Code

`archived/code/` contains the legacy notebooks and one older logistic-regression trainer kept for reference. `archived/bbc/` contains legacy generated figures/GIFs that were previously at the repository root. The active local pipeline now lives in `scripts/`.

`archived/local/` is intentionally ignored. Use it for private local code pulls such as SageMaker notebooks that may contain credential literals. Keep that material private or redact it before making any tracked copy.
